## **:computer: TECNOLOGIAS**


#### **Website** ([React][react])

  - **React Router Dom**
  - **React Icons**
  - **Axios**
  - **Redux**
  - **Redux Thunk**

#### **Server** ([NodeJS][node])

  - **Express**
  - **CORS**
  - **Mongoose e MongoDB Atlas**
  - **dotENV**
  - **Multer**
  - **Yup**
  - **Nodemailer**
